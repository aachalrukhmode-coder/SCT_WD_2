# SCT_WD_2
##Calculator Web Application

This project is a responsive calculator built using **HTML, CSS and JavaScript** as part of the SkillCraft Technology Web Development Internship.

### Features
- Modern calculator interface with responsive design
- Supports basic arithmetic operations (addition, subtraction, multiplication, division)
- Interactive buttons with hover effects
- Event listeners for both button clicks and keyboard input
- Error handling for invalid or incomplete expressions
- DOM manipulation for real-time updates

### Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

### Project Preview
Open the `index.html` file in your browser to try the calculator.
